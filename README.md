# Retail Sales & Inventory Intelligence Platform

Enterprise-grade retail analytics platform built on Google Cloud Platform.

## Architecture

This project demonstrates:

- Real-time retail sales ingestion
- Inventory intelligence
- Medallion architecture
- BigQuery optimization
- Streaming analytics
- Cloud Composer orchestration
- Looker dashboards
- Vertex AI forecasting

## GCP Services Used

- BigQuery
- Pub/Sub
- Cloud Storage
- Cloud Composer
- Dataform
- Looker
- Vertex AI
- Cloud Monitoring
- Dataplex

## Project Structure

- infra/ → Infrastructure setup
- bronze/ → Raw ingestion layer
- silver/ → Cleaned layer
- gold/ → KPI layer
- streaming/ → Real-time ingestion
- composer/ → DAG orchestration
- looker/ → Dashboard definitions
- vertex_ai/ → ML forecasting
- governance/ → Security and policies

## Features

- Real-time dashboard refresh
- Inventory alerts
- Sales forecasting
- Customer segmentation
- Cost-optimized BigQuery design
